CONSIDERACIONES DE EJECUCIÓN

A considerar: el archivo proyecto "Tarea 2" fue creado en Visual Studio Code.

Para que funcionen las distintas opciones del menú, debe ingresar primero la opcion 1: Importar Pokemones, sino no podrá acceder a otras funciones.

Para valores no númericos el programa presenta bucles infinitos en el menú, dado que no corresponde el tipo de opción (está descrito como variable numérica).

Para agregar Pokemones no olvidar que el formato de las variables a Ingresar es capitalizado por lo que por ejemplo un pokemon debe llamarse Blaziken y no blaziken. Para tipos dobles o superiores ingrese Tipo1 (espacio) Tipo2 para que sea ingresado correctamente. Si no tiene evolución previa o siguiente simplemente ingrese N.

Para búsqueda se aplica lo mismo, deben ser ingresados los parámetros de búsqueda exactamente como es requerido al agregar un pokemon.

En ejecución, si se seleccionan más de una vez evolucionar pokemon, es posible que ocurra un segmentation fault que no pudimos encontrar la causa.
